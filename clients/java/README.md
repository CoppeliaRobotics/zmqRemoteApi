# Java ZMQ Remote API

This project contains ZMQ Remote API client for Java.

## Requirements

 - **Java 1.8+**
 - **Maven**
 - **[JeroMQ](https://github.com/zeromq/jeromq)**
 - **[cbor-java](https://github.com/c-rack/cbor-java)**

(maven will automatically download the other dependencies)

## Building:

```
mvn package
```
